# Task 1: Data Overview and Simple Sorting

## Objective

The objective of this task is to understand the structure of the Telco Customer Churn dataset and perform basic data exploration using sorting and filtering techniques in Excel.

---

# Dataset Overview

**Dataset Name:** Telco Customer Churn Dataset

The dataset contains information about customers, their services, account details, billing information, tenure, and churn status.

### Key Columns

| Column Name     | Description                          |
| --------------- | ------------------------------------ |
| CustomerID      | Unique customer identifier           |
| Gender          | Customer gender                      |
| SeniorCitizen   | Senior citizen status                |
| Partner         | Whether customer has a partner       |
| Dependents      | Whether customer has dependents      |
| Tenure          | Number of months customer has stayed |
| PhoneService    | Phone service subscription           |
| InternetService | Internet service type                |
| Contract        | Contract type                        |
| MonthlyCharges  | Monthly billing amount               |
| TotalCharges    | Total amount charged                 |
| Churn           | Customer churn status                |

---

# Understanding Dataset Structure

After opening the dataset in Excel, the following observations were made:

* Rows represent individual customers.
* Columns represent customer attributes and service information.
* The dataset contains demographic, account, and service-related details.
* The Churn column is the target variable indicating whether a customer left the company.

---

# Data Sorting

## Sorting by Tenure

The dataset was sorted in ascending order using the Tenure column.

### Purpose

To identify:

* Customers with the shortest tenure.
* Customers with the longest tenure.
* Customer retention patterns.

### Observation

Customers with very low tenure values are typically new customers and may be at a higher risk of churn.

Customers with high tenure values generally show long-term loyalty and customer retention.

---

# Data Filtering

## Filter Applied

The dataset was filtered using:

**Churn = Yes**

### Purpose

To display only customers who have left the company.

### Observation

The filtered dataset provides valuable information for identifying patterns among churned customers.

Possible factors influencing churn include:

* Higher monthly charges
* Month-to-month contracts
* Short customer tenure
* Limited service subscriptions

# Findings

### Finding 2

### Finding 3

### Finding 4

Tenure appears to be an important factor in understanding customer retention.
